Start simply:

git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle

Add the two vundle lines to your ~/.vimrc file:

set rtp+=~/.vim/bundle/vundle/
call vundle#rc()

Then the “bundles” are as easy as running :BundleInstall from the Vim command prompt.

What are these plugins?

Gist.vim

Add Gists from Vim, selections, full file, private, etc. Includes editing an existing Gist (webapi-vim is a dependency for Gist.vim).

vim-surround

Replace surrounding delimiters, i.e. turn "hello, world", into 'hello, world'. This one is a little complicated, it uses commands, not the command prompt, and there are a lot of options. Check out the readme at the above link.

ctrl-p

This is a fuzzy and MRU (most recently used) search tool for Vim. Press ctrl+p to start searching, highlight a file, then press ctrl+t to open the file in a new tab, for example.

Tabular

This one aligns things like Ruby really wants you to:

{
  :these => 'things',
  :things => 'are',
  :not => 'aligned'
}

:Tab /=> yields the expected result:

{
  :these  => 'things',
  :things => 'are',
  :not    => 'aligned'
}
